# -constellation-art-ndeye-mbisssane-gningue
Ce projet a ete realisedans le cadre de l'evalution de pythonavec le module turtle.L'objectif est de tracer une constellation d'etoile de maniere automatique.

## Declaration du projet
le programme utilise les fonctionnalites de base de 'turtle' pour:
-Creer une fenetre de dessin avec arriere-plan noir (l'espace).
-Placer des points jaunes pour representer les etoiles a des coordonnees precises.
-Relier ces points entre eux avec des lignes colores pour former le destin de la constellations.

## Fichiers inclus
-'projet.py' : le code source principale contenant la logique du destin turtle.
-'README.md' : Ce fichier contenant les exploitations du projet.
